bienvenue au jeu de cliques


Le Jeu de cliques est un jeu simple et addictif dans lequel le joueur doit cliquer sur une boîte qui apparaît aléatoirement sur l'écran pour marquer des points. Le but du jeu est de marquer le plus de points possible en 10 secondes.






le mode de jeu


Cliquez sur la boîte pour marquer des points. 1-La boîte apparaîtra aléatoirement sur l'écran après chaque clic. 2-Vous disposez de 10 secondes pour marquer un maximum de points. 3-Si vous battez votre meilleur score, un message de félicitations s'affichera. 4-Vous pouvez rejouer en cliquant sur le bouton "Rejouer". Objectif : Marquer le plus de points possible en 10 secondes. Contrôles : Cliquez sur la boîte pour marquer des points. Utilisez le bouton "Rejouer" pour rejouer.
